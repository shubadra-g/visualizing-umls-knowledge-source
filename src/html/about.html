UMLS METATHESAURUS

Visual Browser

 About | How To




UMLS Metathesaurus Visual Browser depicts concept relations for terms present in the Metathesaurus. Data is sourced from Unified Medical Language System by National Library of Medicine (NLM), under a license. UMLS has three knowledge sources and one of the is the Metathesaurus. The dataset used is 2018AA.

Motivation Behind the Developing Visual Browser
The current interface for Metathesaurus consists of flowcharts and tabular data. However, these representations score low on ease of use. The language and acronyms in the thesaurus require a certain level of domain knowledge - to use the interface and gather results. This project is a visual browser for the UMLS Metathesaurus and aims to increase the user base as well as present data in the thesaurus in an interactive format.
The corresponding concept relations for the search term are displayed. The 'How To' page has more details on using this browser.
